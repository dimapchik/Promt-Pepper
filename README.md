# Prompt-Pepper
Шеф-ассистент для создания подходящих рецептов на осове ваших предпочтений и содержимого холодильника. С его помощью можно не думать что приготовить вечером, а просто спросить. Prompt-Pepper предложит несколько рецептов из того, что есть у вас дома.
В основе лежит модель gemma2, + RAG на датасете рецептов AkashPS11/recipes_data_food.com.
Анализ метрик:
В качестве судьи мы использовали большую модель llama3.1, в файле rag.ipynb улучшенная модель показала себя лучше обычной.
Для пользования просто перейдите по ссылке https://t.me/prompt_and_pepper_bot и пользуйте на здоровье
